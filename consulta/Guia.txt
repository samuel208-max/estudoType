// ==================================================
// 🚀 CONFIGURAÇÃO COMPLETA DO AMBIENTE TYPESCRIPT
// ==================================================

// 📦 PRIMEIRO: INSTALAR NODE.JS (pré-requisito)
// Acesse: https://nodejs.org → Baixe versão LTS → Instale

// 🔧 COMANDOS PARA EXECUTAR NO TERMINAL (um por vez):

// 1. Criar pasta do projeto e entrar nela
mkdir meu-projeto-typescript
cd meu-projeto-typescript

// 2. Iniciar projeto Node.js (cria package.json)
npm init -y

// 3. Instalar TypeScript como dependência de desenvolvimento  
npm install typescript --save-dev

// 4. Criar arquivo de configuração do TypeScript
npx tsc --init

// 5. Verificar se tudo instalou corretamente
npx tsc --version



// ==================================================
// 🎯 COMANDOS DO DIA A DIA (desenvolvimento)
// ==================================================

// 💻 TERMINAL 1 (deixe rodando):
// Compilação automática - observa mudanças nos arquivos .ts
npx tsc --watch

// 💻 TERMINAL 2 (execute quando quiser testar):
// Executa o código JavaScript gerado
node nome-do-arquivo.js



// ==================================================
// 📁 ARQUIVOS CRIADOS E SUAS FUNÇÕES
// ==================================================

// 📁 node_modules/ - TODAS as bibliotecas instaladas
// 📄 package.json - "RG" do projeto, lista dependências
// 📄 package-lock.json - Trava versões exatas das bibliotecas  
// 📄 tsconfig.json - Configurações do compilador TypeScript
// 📄 *.ts - Seus códigos TypeScript (você edita)
// 📄 *.js - Códigos JavaScript gerados (NÃO edite)



// ==================================================
// 🛠️ FLUXO DE TRABALHO PRÁTICO
// ==================================================

// 1. Crie arquivo .ts (ex: programa.ts)
// 2. Escreva código TypeScript
// 3. Salve → O --watch compila automaticamente para .js
// 4. Execute: node programa.js
// 5. Repita para quantos arquivos quiser!



// ==================================================
// ❌ SOLUÇÃO DE PROBLEMAS COMUNS
// ==================================================

// Erro "comando não encontrado":
// SEMPRE use npx antes: npx tsc (não só tsc)

// Erro com arquivo .js:
// Você tentou compilar JavaScript → Use: npx tsc arquivo.ts

// Watch não funciona:
// Feche terminal → Abra novo → Execute npx tsc --watch novamente



// ==================================================
// 💡 DICAS IMPORTANTES
// ==================================================

// • Sempre use --watch durante desenvolvimento
// • tsconfig.json permite compilar MÚLTIPLOS arquivos .ts
// • node_modules/ NÃO versionar no Git (é muito grande)
// • package-lock.json SIM versionar (garante compatibilidade)
// • O --watch compila TODOS os .ts automaticamente
